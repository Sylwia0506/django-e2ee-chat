# Requirements
- nodejs
- yarn
- poetry

# Installing dependencies
It's mandatory that you install python dependencies before node dependencies.
```shell
poetry install
yarn install
```

# Building
```shell
yarn build
```

# Running
```shell
yarn start
```

# Development
```shell
yarn dev
```
